Sobre el Proyecto

Bibliandria es una aplicación web desarrollada como práctica para el módulo de Diseño de Interfaces del ciclo de Desarrollo de Aplicaciones Web (DAW).

El objetivo principal del proyecto es realizar un estudio completo de:

  • Responsividad: Adaptación de la interfaz a diferentes dispositivos y tamaños de pantalla.
  • Usabilidad: Facilidad de uso, experiencia de usuario y diseño centrado en el usuario.
  • Accesibilidad: Cumplimiento de estándares web para usuarios con diferentes capacidades.

La aplicación permite gestionar bibliotecas personales de libros, con funcionalidades de catálogo, reseñas, préstamos y listas de deseos.
